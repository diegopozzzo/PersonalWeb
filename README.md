# Diego Portfolio — Elegant & Strategic (Next.js + TypeScript)

This project is a TypeScript Next.js portfolio oriented around a **Systems Architect** narrative:
- Prioritizes **NONHUMAN**, **Magnus (Fellow)**, and your role as a connector across domains
- Removes DoersFund; adds a **Pre-seed** section (helping early teams ship robust MVPs)
- Presents Forecast and Trueke as **within the Magnus ecosystem** (brief + strategic)
- Elegant motion: Framer Motion reveals, subtle hover lift, shimmer, floating gradients, refined grid

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Edit content
- Text, nav, affiliations, projects: `src/lib/data.ts`
- Page sections: `src/app/page.tsx`
- Styles: `src/app/globals.css`

## Notes
- Cursor follower is subtle and disabled on touch devices.
